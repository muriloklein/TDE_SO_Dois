# Simulador de Alocação de Blocos

## Objetivo

Demonstrar as três principais formas de alocação de blocos lógicos:

- Contígua
- Encadeada
- Indexada

## Funcionalidades

- Criação e exclusão de arquivos.
- Visualização do disco lógico.
- Seleção do método de alocação.
